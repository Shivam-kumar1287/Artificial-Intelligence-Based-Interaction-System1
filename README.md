# Interaction AI - Multi-Modal AI System

Interaction AI is a premium, context-aware interaction system built using the MERN stack. It allows users to interact with an AI assistant through multiple modalities, including text and voice, while maintaining context and adapting to user preferences over time.

## ✨ Features

- **💬 Text Interaction**: Real-time chat with a clean, glassmorphism interface.
- **🎙️ Voice Integration**: Seamlessly switch to voice input (Speech-to-Text) and enable AI voice output (Text-to-Speech).
- **🧠 Contextual Memory**: The system remembers past interactions to provide more relevant and continuity-focused responses.
- **👤 Personalization**: AI adapts its tone and interaction style based on user behavior and preferences stored in MongoDB.
- **⚡ Real-Time**: Powered by Socket.io for instantaneous communication.
- **💎 Premium UI**: Modern "Nebula" design with smooth animations and responsive layouts.

## 🛠️ Technology Stack

- **Frontend**: React (Vite), Framer Motion, Lucide React, Socket.io-client.
- **Backend**: Node.js, Express, Socket.io, Mongoose.
- **Database**: MongoDB.
- **AI**: Integrated AI Service (extensible to OpenAI/Groq).

## 🚀 Key Requirements Addressed

- **Understand Context**: Maintains history and intent across messages.
- **Multi-modality**: Supports both text and voice.
- **Adaptive**: Learns user preferences for tone and detail level.
- **Responsive**: Real-time interaction loop.
